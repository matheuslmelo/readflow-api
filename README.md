# ReadFlow API

ReadFlow API is a backend project built with Spring Boot for studying and managing book tracking features.

This project is currently in early development and will evolve into a REST API for managing books and reading progress.

## Purpose
- Practice backend development with Java and Spring Boot
- Build a real-world API project for portfolio

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
